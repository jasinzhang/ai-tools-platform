## 分析维度说明

- **买量转化率** ⭐⭐⭐⭐⭐：用户在TikTok/Facebook看到后点击的意愿

## 📊 工具优先级矩阵

```
高转化 + 高停留 = 核心工具（社交媒体生成器）
高转化 + 低停留 = 引流工具（二维码、密码生成）
低转化 + 高停留 = 深度工具（简历优化、产品名称）
低转化 + 低停留 = 补充工具（文本计数、时间戳）
```
