# ⚡ 快速配置代理

## 🚀 3步完成配置

### 步骤1: 打开 .env 文件

在项目根目录找到 `.env` 文件并打开。

### 步骤2: 添加代理配置

在文件末尾添加一行（根据你的代理地址修改）：

```env
HTTPS_PROXY=http://127.0.0.1:7890
```

**常见代理端口**：
- Clash: `7890`
- V2Ray: `10808` 或 `10809`
- 其他代理：查看你的代理软件设置

**如果需要用户名密码**：
```env
HTTPS_PROXY=http://username:password@proxy.example.com:8080
```

### 步骤3: 重启服务器

```bash
# 停止当前服务器（如果正在运行）
# 按 Ctrl+C

# 重新启动
npm start
```

## ✅ 验证配置

启动服务器后，你应该看到：
```
✅ 代理已配置: http://***@127.0.0.1:7890
```

然后测试AI工具，应该可以正常工作了！
