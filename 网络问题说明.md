# ⚠️ Gemini API 网络连接问题

## 🔍 问题诊断

你遇到的错误：`Network error: Unable to connect to Gemini API`

**原因**：无法连接到 Google 的 API 服务器，请求超时。

## 💡 可能的原因

1. **网络环境限制** - 可能无法直接访问 Google 服务
2. **需要 VPN/代理** - Google API 服务器可能被限制访问
3. **防火墙阻止** - 本地防火墙可能阻止了连接

## ✅ 解决方案

### 方案1: 使用 VPN/代理（推荐）

如果你在中国大陆或其他受限网络环境：
1. 启用 VPN 连接到国际网络
2. 或者在服务器上配置代理

### 方案2: 测试非AI工具（立即可用）

以下工具**不需要网络连接**，应该可以正常使用：

- ✅ **密码生成器**: http://localhost:3000/tools/password-generator.html
- ✅ **二维码生成器**: http://localhost:3000/tools/qr-generator.html  
- ✅ **配色方案生成器**: http://localhost:3000/tools/color-palette.html

这些工具是完全本地运行的，不依赖外部API。

### 方案3: 配置代理（如果有）

如果有可用的代理，可以在启动服务器前设置环境变量：

```bash
export HTTP_PROXY=http://your-proxy:port
export HTTPS_PROXY=http://your-proxy:port
npm start
```

### 方案4: 使用其他网络环境

- 使用支持国际网络的网络环境
- 或者使用云服务器（如 AWS、DigitalOcean 等）
